# Guusvanmeerveld's dotfiles

## Install

```bash
./dotfiles install
```

## Add a package to the install list

```bash
./dotfiles add {package_name}
```

example:

```bash
./dotfiles add ungoogled-chromium
```

## Edit a config file

```bash
./dotfiles edit {file_name}
```

example:

```bash
./dotfiles edit bspwmrc
```
